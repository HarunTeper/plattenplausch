# 05 — General idea brainstorm

**Size:** — (ideation) · **Depends on:** — · **Status:** 🟢 anytime

## Goal
Open-ended brainstorm of where Plattenplausch could go — best done as its own focused session
(use the brainstorming skill) so big ideas land *before* #02's architecture sets.

## Seed ideas (unfiltered — to react to, not commitments)
Game / engagement:
- **Scoring depth** (feeds #04): bonus for upsets, captain/double-points pick, bench, weekly MVP.
- **Mini-leagues** — private groups within the overall table (friends-vs-friends).
- **Per-matchday view** — not just season totals; "who won this week".
- **Transfers / wildcards** — though current design is deliberately set-and-forget per round.

Community / podcast tie-in:
- **Shareable team cards** (image export of your lineup for socials).
- **Podcast shoutouts** — top teams read on the show; a field for "podcast handle".
- **Predictions / side bets** alongside the fantasy team.

Operations / trust:
- **Organizer dashboard** — see submissions, send the weekly update, trigger the importer.
- **Email deliverability** — dedicated sending address / domain (already noted as a TODO).
- **Public "how scoring works" page** — once #04's rule is fixed.

Data / seasons (feeds #02):
- **Hall of fame / past champions** across seasons once multi-season exists.
- **Player popularity stats** — most-picked players (anonymous, aggregate).

## How to run it
Invoke the brainstorming skill in a dedicated session; pick a theme (game depth vs community vs
ops) rather than trying to cover all at once. Capture outcomes back into this roadmap (new
workstream files as ideas graduate).
